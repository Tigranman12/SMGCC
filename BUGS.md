# SMGCC Bug Log

This document tracks regressions, edge cases, and root causes. Follow the bug loop rule: stop, document, add a test, fix, re-run everything, then continue.

---

## Format

For each bug entry:
- **Date**: When the issue was discovered
- **Description**: What happened
- **Root Cause**: Why it happened
- **Fix**: How it was fixed
- **Regression Test**: The test added to prevent recurrence
